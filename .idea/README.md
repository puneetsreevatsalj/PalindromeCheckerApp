# Palindrome Checker Application

## Objective
The objective of this project is to develop a Java application that checks whether a given word or string is a palindrome.

## Project Flow
1. User enters a string.
2. The program receives the input.
3. The string is reversed using Java logic.
4. The reversed string is compared with the original string.
5. If both strings match, the program prints "Palindrome".
6. Otherwise, it prints "Not a Palindrome".

## Use Case
This application can be used for:
- Learning basic Java programming
- Understanding string manipulation
- Demonstrating simple algorithm logic

## Project Procedure
1. Created a Java project in IntelliJ IDEA.
2. Implemented a class to check palindrome logic.
3. Used string comparison to verify palindrome.
4. Tested the program with different inputs.
5. Integrated Git for version control.
6. Uploaded the project to GitHub.
